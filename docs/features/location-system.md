# Location System

I believe in using normed systems where possible. Teleportation is the perfect example. One configuration for teleportation applied everywhere.

The crown location system is pretty simple.

You are able to create, edit and delete locdations in-game using the /location command.

In the Core's config.yml you find configuration for the teleportation like the delay, instant teleport regions and worlds aswell as styling.

This system is used a cross my plugins like CrownSpawn, CrownWarps, CrownHomes, CrownCrash and more.
