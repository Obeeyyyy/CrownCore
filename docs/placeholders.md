# Placeholders

This are the plugins placeholders that work with the [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/).

| Placeholder    | Description                                                                                                                                                  |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| %cc\_prefix%   | Returns the configured core prefix.                                                                                                                          |
| %cc\_white%    | Returns the configured core white color.                                                                                                                     |
| %cc\_accent%   | Returns the configured core accent color.                                                                                                                    |
| %cc\_playtime% | Returns the `%statistic_seconds_played%` placeholder for the player, formatted following the configured format in `time-formats.playtime` from `config.yml`. |
